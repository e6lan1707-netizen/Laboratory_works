# Lab 12 — Аутентификация и работа с токенами (JWT)

## *Тема и цель работы*
Изучение механизмов аутентификации и авторизации в веб-приложениях.  
Закрепление навыков работы с токенами (JWT), хранения данных в браузере и защиты доступа.

## *Краткое описание программы*
Приложение реализует:
- форму входа (e-mail и пароль);
- генерацию тестового JWT-токена;
- сохранение токена в localStorage;
- автоматический вход при перезагрузке страницы;
- кнопку "Выход", удаляющую токен и возвращающую пользователя к форме логина.

Программа написана на чистом JavaScript без фреймворков.

## *Как работает*
1. Пользователь вводит e-mail и пароль.
2. Скрипт создаёт JWT (header.payload.signature).
3. Токен сохраняется в localStorage.
4. При повторном заходе приложение автоматически загружает “личный кабинет”.
5. По нажатию “Выйти” токен удаляется.

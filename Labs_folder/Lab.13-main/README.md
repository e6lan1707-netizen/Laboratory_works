Отчёт по лабораторной работе «Веб-приложение для работы с коллекцией песен (музыкальный плеер)»

1. Назначение и цель работы

Целью лабораторной работы является разработка простого веб-приложения для работы с коллекциями данных, основанного на HTML, CSS и JavaScript без использования внешних библиотек.
В рамках задания реализовано приложение для управления коллекцией песен с возможностью добавления, редактирования, удаления и сохранения данных.

Приложение выполнено в стиле минималистичного гранж-дизайна, подчёркивающего творческую направленность проекта.

⸻

2. Функциональные возможности

Разработанное веб-приложение поддерживает следующие функции:

✔ Добавление новой песни

Пользователь может указать:
- название песни
- исполнителя
- год
- жанр
- ссылку на обложку альбома (URL)
- ссылку на аудиофайл (URL)

✔ Удаление песен

Каждая запись может быть удалена нажатием соответствующей кнопки.

✔ Поиск песен 

Можно найти песню по параметру: название, исполнтель, год выпуска, жанр.

✔ Редактирование данных

Нажатие кнопки «редактировать» переносит данные выбранной записи обратно в форму.

✔ Встроенный аудиоплеер

Поддерживается воспроизведение аудио по прямой ссылке.

✔ Сохранение данных

Коллекция автоматически сохраняется в localStorage.
После перезагрузки браузера данные сохраняются.

✔ Визуальная часть
- адаптированный тёмный гранж-стиль
- текстурированный фон
- тёмная палитра, красно-серые акценты
- минималистичная таблица
- оформление форм и кнопок в едином стиле

⸻

3. Используемые технологии

При реализации проекта использованы стандартные средства веб-разработки:
 - HTML5 — структура приложения
 - CSS3 — оформление интерфейса
 - JavaScript (ES6) — управление коллекцией, DOM-операции, логика редактирования, удаления и сохранения
 - LocalStorage API — хранение данных на стороне пользователя

Внешние библиотеки и фреймворки не применялись.

⸻

4. Структура приложения

Файл состоит из трёх основных частей:

1) HTML-разметка
- форма добавления песен
- таблица коллекции
- встроенный аудиоплеер

2) CSS-стили
- тёмная эстетика с текстурами
- оформление таблицы, кнопок и формы
- акценты на элементах управления

3) JavaScript-логика
 - массив songs для хранения записей
 - функции render(), save(), deleteSong(), editSong(), playSong()
 - привязка событий к кнопкам и форме

⸻

6. Вывод

В ходе выполнения лабораторной работы было создано полностью рабочее веб-приложение для управления коллекцией песен (или же по просту музыкальный плеер).
Реализованный функционал удовлетворяет требованиям задания, а также расширен дополнительными возможностями: встроенный аудиоплеер, поддержка изображений и использование localStorage.
Проект является автономным, легко переносимым и не требует серверной части или сторонних библиотек.
